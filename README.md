  #    Git 命令
1. 先将别人的项目进行克隆下来
git clone https://github.com/opentianwe/XUEonline
 
2.在克隆的基础上进行修改文件，如我修改README.md
 

3. 将改变后的文件添加到git上传仓库列表中
git add README.md
 
4. 提交本次修改的说明
git commit -m "message"
 

5. 推送到远端
git push origin mastergit 

也可以直接用git push

.二次提交说明

git commit -m "修改说明"
